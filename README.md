# Proyecto SIGPD 

Este proyecto es parte del Sistema Informático de Gestión de Partidas para Draftosaurus (SIGPD). La aplicación permite al usuario colocar fichas de colores en 7 zonas de un tablero, simulando la mecánica del juego adaptado al entorno web.

## 🗂 Estructura del repositorio

Proyecto-SIGPD/
├── index.html # Página principal con las zonas y fichas
├── estilo.css # Estilos visuales, animación de fondo
├── script.js # Funcionalidad de arrastrar y soltar fichas
└── img/
    └── fondo-sin-letras.jpg # Imagen animada de fondo

## 🌐 Tecnologías usadas

- HTML5
- CSS3 (con animaciones)
- JavaScript (drag and drop)
