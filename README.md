# S.I.G.P.D. (Sistema Informático de Gestión de Partidas para Draftosaurus)



## 📌 Descripción del Proyecto
Este proyecto forma parte de una solución digital para gestionar partidas del juego **Draftosaurus**.  A través de una interfaz interactiva, el usuario puede colocar fichas de colores en 7 zonas distintas del tablero, simulando las dinámicas del juego físico dentro de un entorno web.

---

## 🕹️ Funcionalidad del Juego

El usuario puede arrastrar y soltar fichas de colores en distintas zonas del tablero (3 a la izquierda, 1 central y 3 a la derecha). Cada zona representa una parte del juego inspirada en Draftosaurus, adaptada a una versión digital. Las fichas simulan dinosaurios y deben ser colocadas según reglas específicas que luego serán evaluadas.

## 🧪 Tecnologías Utilizadas

- **HTML5**: estructura de la página
- **CSS3**: diseño visual y animación de fondo
- **JavaScript (ES6)**: lógica de interacción drag and drop
- **Editor.md**: editor Markdown para documentación

---

## 🧠 Lógica Implementada

- Sistema de fichas drag & drop nativo con JavaScript
- Cada zona es un contenedor receptivo con validación de fichas
- Animación suave del fondo mediante `@keyframes` y `background-position`
- Código modular separado en HTML, CSS y JS

## 🧩 Reglas del Juego

Cada zona tiene reglas distintas sobre cómo se colocan las fichas de colores (representando dinosaurios). Las reglas están inspiradas en Draftosaurus.
